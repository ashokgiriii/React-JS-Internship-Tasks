What is Responsive Design ?

Responsive web design (RWD) is a web development approach that creates a single, flexible website that automatically adapts to different screen sizes, platforms, and orientations. This ensures that the site is visually appealing and functional whether it's viewed on a desktop, tablet, or smartphone, without the need for separate versions for each device.

Rule of Responsive

  • Use rem/em for breakpoints & very small fixed things.
  • Use rem/em for text, spacing, padding, margins.
  • Use % / vw / vh for layout widths/heights.
